📌 Projetos em destaque

### Ponto Digital Sync8
Sistema interno de ponto, banco de horas e gestão de acesso.

Projeto desenvolvido por mim com foco em controle de jornada, autenticação segura, perfis de acesso e organização administrativa para equipes.

🔗 Acesse: [pontodigital-sync8.vercel.app](https://pontodigital-sync8.vercel.app/)

**Principais recursos:**
- Login com e-mail e senha
- Autenticação com hash bcrypt
- Troca de senha
- Perfis por permissão: admin, gestor e colaborador
- Controle de ponto e banco de horas
- Gestão de acesso administrativo
